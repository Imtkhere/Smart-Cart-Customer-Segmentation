# SmartCart Customer Segmentation

## Overview

SmartCart Customer Segmentation is an Unsupervised Machine Learning project that groups customers based on their purchasing behavior and demographic information. The goal is to help businesses identify different customer segments for personalized marketing and better decision-making.

---

## Project Objectives

- Perform customer segmentation using Machine Learning.
- Analyze customer behavior based on spending patterns.
- Identify similar customer groups.
- Visualize customer clusters using PCA.
- Compare clustering results for business insights.

---

## Dataset Features

The dataset contains customer information such as:

- Year of Birth
- Education
- Marital Status
- Income
- Kidhome
- Teenhome
- Customer Joining Date
- Recency
- Wine Spending
- Fruit Spending
- Meat Spending
- Fish Spending
- Sweet Spending
- Gold Product Spending
- Number of Deals Purchased
- Web Purchases
- Catalog Purchases
- Store Purchases
- Monthly Website Visits
- Response
- Complaint
